INKU v13 — START HERE

1. Open Android Studio.
2. Choose File -> Open.
3. Select this Inku folder (the folder containing app, gradle, gradlew and settings.gradle.kts).
4. Choose Trust Project if asked.
5. Wait for Gradle Sync to finish.
6. Select your Samsung phone and the app run configuration.
7. Press Run.

FIRST RUN
- Choose one main folder when Inku asks.
- Inku creates: Inku/Downloads, Local Anime, Local Manga, Subtitles, Covers,
  Backgrounds, Thumbnails, Backups and Extensions.
- Local titles appear under Browse -> Local Anime / Local Manga first.
- A title only appears in the Anime or Manga library after you press Add to library.

LOCAL ANIME EXAMPLE
Inku/Local Anime/Demon Slayer/
  cover.jpg
  background.jpg
  metadata.json
  Season 1/
    Episode 01.mp4
    Episode 02.mkv

PLAYBACK
- Open a local anime from Browse or your library.
- Press Play / Continue or tap an episode thumbnail.
- Subtitles are selected inside the Inku player.

DOWNLOADS
- Local episode downloads are real file copies into Inku/Downloads/Anime/<title> - EN.
- Network downloads only work when an installed Inku source supplies a permitted direct file URL.
